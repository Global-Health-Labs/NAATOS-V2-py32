# py32
Installs enviornment configuration required to BUILD, FLASH, DEBUG with PUYA32 

Run with parameter -AcceptSEGGARLicense

# Tools
- installs make
- installs curl
- installs chocolatey


# Libraries
- installs JLINK (SEGGAR)
- installs PUYA32 device list for JLINK (not in default installation), included in ..\PY32,
	installation location: $env:USERPROFILE\AppData\Roaming\SEGGER\JLinkDevices"

- installs arm-none-eabi-gcc (ARM GCC tools)
